## User flow (iPhone PWA)
1. Open gymbro (port 7000, Tailscale alonso)
2. Bottom nav tab 我嘅飲食 or hero card 今日總結下嘅「+掃描食物」按鈕
3. iPhone camera opens OR file picker for receipt screenshot
4. Snap / select → POST `/api/scan_food` (multipart image)
5. Backend runs vision + pplx + share-locked-macro reconcile
6. Confirmation card appears with: 餐名 / 餐廳 / kcal / P / C / F / 來源 / 時間
7. (Optional) 「更正」按鈕 → POST `/api/scan_correct` with Jim's text correction; re-feed model for future precision

## Implementation recipe
```python
@app.route('/api/scan_food', methods=['POST'])
def api_scan_food():
    if 'image' not in request.files:
        return jsonify(error='no image'), 400
    img_bytes = request.files['image'].read()
    
    # 1. MiniMax M3 vision (base64 inline)
    b64 = base64.b64encode(img_bytes).decode()
    payload = {
        "model": "MiniMax-Text-01",
        "messages": [{"role": "user", "content": [
            {"type": "text", "text": "詳細描述呢張食物相。逐樣列菜式、份量、煮法、醬汁、容器。係咪見到 logo/招牌字?估計每樣卡路里同蛋白質。繁中廣東話。"},
            {"type": "image_url", "image_url": {"url": f"data:image/jpeg;base64,{b64}"}}
        ]}],
        "max_tokens": 1800, "temperature": 0.25
    }
    # POST https://api.minimax.io/v1/chat/completions (image+vision path)
    desc = call_minimax_vision(payload)
    
    # 2. Pplx sonar-pro enrichment
    pplx_prompt = f"由以下食物描述:「{desc}」, lookup 餐廳/連鎖/品牌 nutrition 同 standard portion 大細, kcal/P/C/F 全列。繁中。"
    enrichment = call_pplx(pplx_prompt)  # 5-10s, ~$0.018
    
    # 3. Apply 60/40 share silently if dish description suggests share scenario
    is_shared = detect_shared_meal(desc)  # heuristic: 「兩人份」「分享」「share」/multiple dishes
    if is_shared:
        jim_kcal = round(total_kcal * 0.60)
        # ...
    else:
        jim_kcal = total_kcal  # 100% Jim
    
    # 4. Append to nutrition_log.json + Sheet Nutrition tab
    entry = build_entry(desc, enrichment, jim_kcal, ...)
    log = json.load(open('/home/work/.hermes/nutrition_log.json'))
    log['meals'].append(entry)
    json.dump(log, open('/home/work/.hermes/nutrition_log.json', 'w'), indent=2, ensure_ascii=False)
    append_to_sheet_nutrition(entry)
    
    return jsonify(entry)
```
